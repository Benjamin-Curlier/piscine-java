# Module 5 — sous-groupe 5.2 (Lecture/écriture de fichiers) Implementation Plan

> **For agentic workers:** Exécution **ultracode**. 1 agent produit 1 exo complet à partir de la spec §2.X ; le **pilote** relance le gate Maven et commit ; clôture pilotée. Checkbox (`- [ ]`) pour le suivi.

**Goal:** Produire les 4 exercices du sous-groupe 5.2 (`compteur-lignes`, `copie-fichier`, `journal-evenements`, `lecture-config`) sous `exercises/module-5-exceptions-io/`, conformes à la spec [`specs/2026-06-04-m5-exos-5-2-design.md`](../specs/2026-06-04-m5-exos-5-2-design.md).

**Architecture:** Premier sous-groupe **I/O** des exos. Méthodes `static` prenant un `Path` fourni par l'appelant et propageant `throws IOException` ; flux `Files.lines` en `try-with-resources` ; **NIO.2 imposé** ; UTF-8 + `\n` explicites. Tests par **`@TempDir`** (JUnit Jupiter, aucune dépendance ajoutée) asserant le **contenu**/retour. Calque de format : exos 5.1 + M4.

**Tech Stack:** Java 25 (`E:/java/jdk-25.0.3+9`), Maven (`E:/java/apache-maven-3.9.9`), JUnit 5.11.0, AssertJ 3.26.3, build-helper. Package `etnc.m5`, groupId `etnc.piscine.m5`.

**Modèle de pom** : copie conforme d'un pom 5.1/M4 (ex. `exercises/module-5-exceptions-io/5.1.1-saisie-defensive/{starter,solution}/pom.xml`), changer groupId/artifactId/name. Starter ajoute `../tests` ; solution ajoute `../tests` + `../tests-prives`.
