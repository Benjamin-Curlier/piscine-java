# Module 5 — sous-groupe 5.1 (Exceptions) Implementation Plan

> **For agentic workers:** Exécution **ultracode** (multi-agents, autorisée explicitement par le formateur pour ce cycle). 1 agent produit 1 exo **complet** à partir de la spec §2.X ; le **pilote** relance le gate Maven entre/après les agents ; clôture (lint + backlog + PR) pilotée. Steps en checkbox (`- [ ]`) pour le suivi.

**Goal:** Produire les 3 exercices du sous-groupe 5.1 (`saisie-defensive`, `effectif-invalide`, `calculs-geometriques-exceptions`) sous `exercises/module-5-exceptions-io/`, conformes à la spec [`specs/2026-06-04-m5-exos-5-1-design.md`](../specs/2026-06-04-m5-exos-5-1-design.md).

**Architecture:** Chaque exo = 9–11 fichiers (format-exercice §2, calque M4). Premier exo du module → **crée le dossier** `exercises/module-5-exceptions-io/`. Nouveauté vs M1-M4 : les solutions **lèvent** des exceptions et les tests les **vérifient** (`assertThatThrownBy` / `getCause` / `assertThatCode`). Le `starter/` impose les signatures à corps vide (compile mais échoue les tests publics). Pas d'I/O (que des exceptions, ch.1-4).

**Tech Stack:** Java 25 (JDK `E:/java/jdk-25.0.3+9`), Maven (`E:/java/apache-maven-3.9.9`), JUnit Jupiter 5.11.0, AssertJ 3.26.3, `build-helper-maven-plugin`. Package `etnc.m5`, groupId `etnc.piscine.m5`.

**Modèle de pom (mécanique, réutilisé partout)** : copie conforme d'un pom M4 (ex. `exercises/module-4-collections-generiques-lambdas/4.1.1-annuaire-militaire/{starter,solution}/pom.xml`). Seuls changements : `<groupId>etnc.piscine.m5</groupId>`, `<artifactId>` = slug (solution : slug + `-solution`), `<name>`. Starter ajoute la seule source `../tests/src/test/java` ; solution ajoute `../tests/...` **et** `../tests-prives/...`.

**Convention de gate local (pilote, à chaque exo)** — depuis la racine du repo :
```bash
# 1) Solution DOIT passer publics + privés
JAVA_HOME="E:/java/jdk-25.0.3+9" "E:/java/apache-maven-3.9.9/bin/mvn" \
  -q -f exercises/module-5-exceptions-io/<dossier-exo>/solution/pom.xml test
# 2) Starter DOIT compiler (tests publics inclus), échouer aux assertions (pas d'erreur de compile)
JAVA_HOME="E:/java/jdk-25.0.3+9" "E:/java/apache-maven-3.9.9/bin/mvn" \
```
