## 4. Les 4 exercices

### 4.3.1 `filtrage-projection` — gabarit — idiomatisme (desc : « pipeline `filter`/`map`/`collect` idiomatique ; lambda en paramètre »)
**Fourni** : `Grade.java` + `Soldat.java`. **À compléter** : `Effectifs`.
```
static List<String> nomsDesGradesAuMoins(List<Soldat> soldats, Grade min)
   // soldats.stream().filter(s -> s.grade().compareTo(min) >= 0).map(Soldat::nom).collect(Collectors.toList())
static List<Soldat> filtrer(List<Soldat> soldats, Predicate<Soldat> critere)
   // soldats.stream().filter(critere).collect(Collectors.toList())  -- Predicate en paramètre (streams imposés par design)
```
**Starter** : `Effectifs` stubs `return null; // TODO` (import `java.util.function.Predicate`).
**Tests publics** (≈3) : `nomsDesGradesAuMoins(liste, SERGENT)` → `containsExactly` les noms des soldats de grade ≥ SERGENT dans l'ordre source ; `filtrer(liste, s -> s.anciennete() > 5)` (lambda passée par le test) → sous-ensemble correct ; `filtrer` avec `s -> s.grade() == Grade.LIEUTENANT`.
**Tests privés** (≈3) : aucun élément ne satisfait → liste vide (pas `null`) ; tous satisfont → tous renvoyés ; predicate toujours faux → vide ; la liste source n'est pas modifiée.

### 4.3.2 `agregation` — demarche (desc : « réductions numériques, cas liste vide géré par valeur-sentinelle »)
**Fourni** : `Grade.java` + `Soldat.java`. **À compléter** : `Statistiques`.
```
static int total(List<Soldat> soldats)                // soldats.size()  (ou .stream().count())
static double ancienneteMoyenne(List<Soldat> soldats) // .stream().mapToInt(Soldat::anciennete).average().orElse(0.0)
static int ancienneteMax(List<Soldat> soldats)        // .stream().mapToInt(Soldat::anciennete).max().orElse(0)
```
**Starter** : stubs `return 0; / return 0.0; // TODO`.
**Tests publics** (≈3) : `total` = nombre d'éléments ; `ancienneteMoyenne` → `isCloseTo(valeur, within(0.001))` ; `ancienneteMax` = plus grande ancienneté.
**Tests privés** (≈3) : liste **vide** → `ancienneteMoyenne` 0.0 (`isCloseTo(0.0, within(0.001))`) et `ancienneteMax` 0 (sentinelles, pas d'exception) ; un seul élément → moyenne = sa valeur ; valeurs égales → moyenne = cette valeur.

### 4.3.3 `groupement-stream` — idiomatisme (desc : « collecteurs `groupingBy`/`partitioningBy`/`counting` »)
**Fourni** : `Grade.java` + `Soldat.java`. **À compléter** : `Regroupement`.
```
static Map<Grade, List<Soldat>> parGrade(List<Soldat> soldats)            // groupingBy(Soldat::grade)
static Map<Boolean, List<Soldat>> selonAnciennete(List<Soldat> s, int seuil) // partitioningBy(x -> x.anciennete() >= seuil)
static Map<Grade, Long> effectifsParGrade(List<Soldat> soldats)           // groupingBy(Soldat::grade, Collectors.counting())
```
**Starter** : stubs `return null; / return Map.of(); // TODO`.
**Tests publics** (≈3) : `parGrade` → `get(SERGENT)` contient les bons soldats (`containsExactlyInAnyOrder`) ; `selonAnciennete` → partitions `true`/`false` correctes ; `effectifsParGrade` → comptes par grade.
**Tests privés** (≈3) : liste vide → `parGrade` map **vide** ; `partitioningBy` garantit **TOUJOURS** les deux clés `true` ET `false` (même groupe vide) — distinction explicite avec `groupingBy` ; un grade unique → une seule clé.

### 4.3.4 `recherche-optional` — exo final — idiomatisme (desc : « `Optional` correctement utilisé : `map`/`orElse`, pas de `get()` nu »)
**Fourni** : `Grade.java` + `Soldat.java`. **À compléter** : `Recherche`.
```
static Optional<Soldat> premier(List<Soldat> soldats, Predicate<Soldat> critere)
   // soldats.stream().filter(critere).findFirst()
static Optional<Soldat> plusHautGrade(List<Soldat> soldats)
   // soldats.stream().max(Comparator.comparing(Soldat::grade))
static String nomOuParDefaut(List<Soldat> soldats, Predicate<Soldat> critere)
   // premier(soldats, critere).map(Soldat::nom).orElse("Aucun")
```
**Starter** : stubs — `return Optional.empty(); / return "Aucun"; // TODO` (compile ; les tests « présent » échouent).
**Tests publics** (≈3) : `premier` **présent** quand un élément satisfait (`assertThat(opt).isPresent()` + `.contains(soldatAttendu)`), **vide** sinon (`.isEmpty()`) ; `plusHautGrade` → `.contains(leSoldatDeGradeMax)` ; `nomOuParDefaut` → le nom, ou `"Aucun"`.
**Tests privés** (≈3) : liste vide → `Optional.empty()` partout ; aucun match → empty ; `nomOuParDefaut` sur liste vide → `"Aucun"` (`orElse`). **Test via `OptionalAssert`** (`isPresent`/`contains`/`isEmpty`/`hasValue`), jamais sur la façon d'obtenir l'`Optional`. `contains` s'appuie sur l'`equals` généré du record `Soldat`.
