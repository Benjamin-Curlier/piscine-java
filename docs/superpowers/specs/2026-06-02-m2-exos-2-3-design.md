## 3. Spécification par exercice

### 2.3.1 — biblio-maths (`BiblioMaths`, `moyen`, 40 min)
Trois méthodes `static` :
- `static int pgcd(int a, int b)` — plus grand commun diviseur (algorithme d'Euclide). Entiers **≥ 0**. `pgcd(0, 5) = 5`, `pgcd(12, 18) = 6`.
- `static boolean estPremier(int n)` — `true` si `n` est premier. **`n < 2` → `false`**. `estPremier(2) = true`.
- `static int sommeChiffres(int n)` — somme des chiffres de `n` (**`n ≥ 0`**). `sommeChiffres(123) = 6`, `sommeChiffres(0) = 0`.

**Tests publics** : `pgcd(12,18)=6` ; `estPremier(7)=true` ; `sommeChiffres(123)=6`.
**Cas privés** : `pgcd(17,13)=1` (coprimes), `pgcd(100,10)=10`, `pgcd(0,5)=5` ; `estPremier(1)=false`, `estPremier(2)=true`, `estPremier(97)=true`, `estPremier(100)=false` ; `sommeChiffres(0)=0`, `sommeChiffres(9999)=36`.

### 2.3.2 — surcharge (`Surcharge`, `facile`, 25 min)
Trois méthodes `static` de **même nom** :
- `static int maximum(int a, int b)`
- `static int maximum(int a, int b, int c)`
- `static double maximum(double a, double b)`

**Tests publics** : `maximum(3,7)=7` ; `maximum(3,7,5)=7` ; `maximum(2.5,1.5)=2.5`.
**Cas privés** : `maximum(-3,-7)=-3` ; égalité `maximum(4,4)=4` ; `maximum(1,9,5)=9` (max au milieu), `maximum(9,1,5)=9` (max en tête) ; `maximum(-1.5,-0.5)=-0.5`.

### 2.3.3 — refactor-calculs-geometriques (`CalculsGeometriques`, `moyen`, 35 min)
Refactor de 1.2.2 (cercle) en méthodes `static`, avec surcharge cercle/rectangle :
- `static double aire(double rayon)` — `Math.PI * rayon * rayon`.
- `static double aire(double largeur, double hauteur)` — `largeur * hauteur`.
- `static double perimetre(double rayon)` — `2 * Math.PI * rayon`.
- `static double perimetre(double largeur, double hauteur)` — `2 * (largeur + hauteur)`.

**Tests publics** : `aire(2.0)` ≈ `Math.PI*2*2` ; `aire(3.0,4.0)=12.0` ; `perimetre(2.0)` ≈ `2*Math.PI*2` ; `perimetre(3.0,4.0)=14.0`.
Pour les cas cercle, l'attendu est **calculé avec la même formule** dans le test (`Math.PI*r*r`), donc comparaison par **égalité exacte** (`isEqualTo`). Pour le rectangle, valeurs exactes.
**Cas privés** : `aire(1.0)=Math.PI` ; `perimetre(1.0)=2*Math.PI` ; `aire(5.0,2.0)=10.0` ; `perimetre(5.0,2.0)=14.0` ; carré dégénéré via rectangle `aire(3.0,3.0)=9.0`.
